# spring-boot-on-docker-compose
<h4> 1. 실행방법</h4>
<h5>1) foreground 실행</h5>

```
$ docker-compose up
```

<h5>2) background 실행</h5>

```
$ docker-compose up -d
```

<h5>3) docker-compose 종료</h5>

```
$ docker-compose down
```
